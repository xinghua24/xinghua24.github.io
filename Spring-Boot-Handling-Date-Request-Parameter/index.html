Handling Date types in the Spring Boot Controller

Handling Date type

Date can be received as parameterusing long, java.utils.Date, ZonedDateTime and LocalDateTime.
The @DateTimeFormat annotation lets you specify the format with the ISO enum. Alternatively, the format can be specified with the pattern attribute of the annotation.

see DateController.java

If you have Jackson integeration with your application to serialize your bean to JSON format
then you can use Jackson anotation @JsonFormat to format you date to specified format.
