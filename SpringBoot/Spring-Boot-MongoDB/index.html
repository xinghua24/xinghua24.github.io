Spring Boot Integration with mongoDB

Maven Dependency

Add dependency for mongodb. Use embed mongodb instance in this project

MongoDB Integration

MongoDB integration is very similar to MySQL integration. we add annotaiton to entity class and then create repository class to interact with the data store.

Employee class

@Document(collection="employee") shows this is a collection in mongodb

EmployeeRepository.java

MongoRepository contains many useful methods to interact with Mongo Collection. We can add additional method here to query the collection.

Besides extending MongoRepository class, you can autowire a MongoTemplate object to write more complex queries.

EmployeeService.java

EmployeeController.java

EmployeeController defines many endpoints to interact with the employee collection.
