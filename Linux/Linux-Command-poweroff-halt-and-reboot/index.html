Linux Command - poweroff, halt and reboot

Posted on 2019-01-08 In Linux-Command

poweroff, halt and reboot are commands to stop system hardward.

  • poweroff sends ACPI signal to tell the system to powerdown
  • halt stops CPU function, but leaves it powered on.
  • reboot reboots the system

The is also a shutdown command lets you schedule a time to halt, poweroff orreboot the machine.
