Creating Executable Jar

To create an executable jar(fat jar)
, we need to have spring-boot-maven-plugin in our pom.xml file. The project generated by Spring initializr adds spring-boot-maven-plugin in pom.xml already.

Run mvn package will generate the executable jar file. use java -jar command to run the jar. The executable jar is ready for production deployment.
